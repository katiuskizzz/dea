Introduccion

Un bosquejo permite planificar la redacción de un proyecto, para organizar las ideas, para luego expresar como se va lograr y que se va realizar, el bosquejo es también la primera traza o diseño que se realiza de una obra pictórica o de cualquier otra producción de la creatividad humana. El bosquejo supone el primer paso para realizar un plan de proyecto, como se va a realizar en el siguiente trabajo que se plasmará algunos bosquejos de una página web de subasta de café que contendrá principalmente el menú, categorías del producto que se va a subastar y el registro de usuario.
